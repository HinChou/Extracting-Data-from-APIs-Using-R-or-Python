This function can extract all tweets data from a certain twitter account.

Because the limit of the tweepy, actually you can all extract around 3200 tweets from that account.

The function returns all these tweets in a well list format.

The function can be used as the first step of the data mining, which is extracting and formating data.



Reference: http://docs.tweepy.org/en/v3.5.0/auth_tutorial.html
